Projet Calculatrice

  J'ai fait ce projet pour le plaisir d'apprendre comment faire une calculatrice en HTML et en CSS.

  Voici les lignes de ce code en HTML:


    <div class="container">
        <div class="calculator">
            <form>
                <div class="display">
                    <input type="text" name="display" readonly>

                <div>
